This project is a Power BI dashboard designed to analyze hospital operations and performance.
It provides insights into patient activity, doctor workload, billing, and appointment trends.

📁 Dataset

The data used in this project includes the following tables:

appointments
patients
doctors
billing
treatments
📈 Dashboard Insights
🔹 Key Metrics (Top Cards)
Total Patients
Total Appointments
Total Revenue
🔹 Appointments Trend

A line chart showing how appointments change over time.
➡️ Helps identify peak periods and demand patterns.

🔹 Revenue by Service / Category

A stacked bar chart displaying revenue distribution.
➡️ Useful for understanding which services generate the most income.

🔹 Top Doctors by Appointments

A bar chart ranking doctors based on number of appointments.
➡️ Highlights workload distribution among doctors.

🔹 Patient Distribution

A pie chart showing how patients are distributed (e.g., categories or types).
➡️ Helps in understanding patient segmentation.

🛠 Tools Used
Power BI
Excel
🎯 Objective

The goal of this dashboard is to support data-driven decision-making in hospital management by:

Monitoring performance
Optimizing resource allocation
Improving patient experience
